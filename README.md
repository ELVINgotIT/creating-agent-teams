## üéØ How to Use creating-agent-teams

### Create Your First Agent

1. Click the **Create Agent** button.
2. Choose if you want a single agent or a team.
3. Name your agent or team.
4. Set the task or goal you want them to complete.
5. Click **Start** to activate.

### Managing Teams

- For tasks needing multiple steps or parallel work, create subagents to handle parts separately.
- Use the dashboard to monitor team progress.
- Stop, pause, or restart agents anytime.

### Tips for Best Results

- Start with single agents for simple tasks.
- Use subagents when tasks can be split ‚Äî this speeds up work.
- Use full teams for large or complex projects needing coordination.
- Monitor agents regularly to ensure they meet your goals.
